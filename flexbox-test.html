10 Preguntas sobre Flexbox

Flexbox es un modelo de diseño en CSS que proporciona una manera flexible y eficiente de organizar y alinear elementos en un contenedor. Sus ventajas incluyen un flujo de diseño más fácil y controlable, distribución automática de espacios en blanco y un enfoque más sencillo para el diseño responsivo.

En Flexbox, el eje principal es la dirección en la que los elementos se distribuyen dentro del contenedor, mientras que el eje transversal es perpendicular al eje principal. El eje principal se define utilizando la propiedad flex-direction.

display: flex; convierte el contenedor en un bloque de nivel de bloque con un ancho que se ajusta al contenido. display: inline-flex; convierte el contenedor en un nivel de línea con un ancho que se ajusta al contenido.

La propiedad flex-direction establece la dirección principal en la que se distribuyen los elementos dentro del contenedor flex. Los posibles valores son row, row-reverse, column y column-reverse.

Para alinear elementos flex en el eje principal, se utiliza la propiedad justify-content, y para alinearlos en el eje transversal, se utiliza la propiedad align-items en el padre o align-self en los propios elementos.

La propiedad flex-grow determina cómo se distribuye el espacio adicional en un contenedor flex entre los elementos flexibles. Se utiliza para especificar qué elementos deben crecer y en qué proporción.

La propiedad flex-shrink define la capacidad de los elementos flexibles para reducir su tamaño cuando el espacio disponible es insuficiente. Se utiliza para especificar qué elementos deben encogerse y en qué proporción.

flex-basis establece el tamaño inicial de un elemento flexible antes de que se distribuya el espacio adicional, mientras que width establece el tamaño fijo de un elemento independientemente del espacio disponible. flex-basis tiene prioridad sobre width cuando se está aplicando flex.

flex-wrap determina si los elementos flexibles deben envolverse o ajustarse dentro del contenedor flex cuando no tienen suficiente espacio en una línea. Sus posibles valores son nowrap, wrap y wrap-reverse.

La propiedad align-content se utiliza para alinear las filas de elementos flexibles cuando hay espacio adicional en el eje transversal. Sus valores incluyen flex-start, flex-end, center, space-between, space-around y stretch.
